# Project 2 — Analysis of processed Hi-C maps

This project analyzes prepared Hi-C contact maps for four samples:

- MoPh7_enr_v2
- MoPh11_enr_v2
- MoPh14_enr_v2
- MoPh15_enr_v2

## Analyses

- Visualization of multi-resolution Hi-C maps
- Contacts vs genomic distance
- Insulation score and domain boundaries
- A/B compartment analysis using E1
- Saddle plots
- Export of E1 tracks as bedGraph and bigWig

## Results

The four samples showed similar Hi-C contact decay profiles and broadly similar A/B compartment organization. Insulation score analysis showed local domain boundaries, with generally comparable patterns between samples.
