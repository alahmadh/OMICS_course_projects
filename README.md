# OMICS course spring 2026

Практические материалы к курсу по методам анализа данных структурно-функциональной
организации хроматина.

Репозиторий организован по дням: каждый день лежит в отдельной папке со своим
`README.md`, данными, результатами, скриптами и ноутбуками.

## Расписание

| Дата | Тема | Материалы |
| --- | --- | --- |
| 04.06 | Hi-C: 3D организация хроматина, уровни организации, эу- и гетерохроматин, Juicer, визуализация карт контактов | лекция подготовлена отдельно |
| 09.06 | Практика Hi-C + RNA-seq и CAGE-seq: подготовка Hi-C ридов, Juicer, визуализация и анализ; выравнивание RNA-seq/CAGE-seq | [day1_HiC_practice](day1_HiC_practice/) |
| 10.06 | ChIP-seq: гистоновый код, MACS3, коллинг пиков, визуализация | будет добавлено |
| 16.06 | Метилирование ДНК: бисульфитное секвенирование, Bismark, разметка регионов с высоким уровнем метилирования | будет добавлено |
| 17.06 | Комплексный анализ различных видов данных | будет добавлено |

## Структура одного практического дня

```text
dayXX_topic/
├── README.md
├── data/
│   ├── raw/
│   ├── trimmed/
│   ├── reference/
│   └── juicer/
├── results/
├── notebooks/
├── scripts/
└── tools/
```

Большие данные, результаты обработки, `.hic` файлы и скачанные инструменты не
коммитятся в git. В репозитории остаются инструкции, ноутбуки, скрипты и небольшие
служебные файлы.
